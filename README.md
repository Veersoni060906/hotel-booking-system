HOTEL-BOOKING-SYSTEM

Structure:
- backend/ (Express + MongoDB)
- frontend/ (React)

Quick start:
1. Backend
   cd backend
   npm install
   edit .env (MONGO_URI)
   npm run seed          # seeds DB from backend/data/*.json
   npm run dev           # start backend (nodemon)

2. Frontend
   cd frontend
   npm install
   npm start

Open http://localhost:3000
